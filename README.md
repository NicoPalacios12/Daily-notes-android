# Daily Notes

Application Android de gestion de tâches avec catégories personnalisées.

## Features
- Créer, éditer et supprimer des tâches
- Catégories avec couleurs personnalisées
- Filtrer par catégorie
- Marquer les tâches comme complétées

## Stack
- Kotlin + Jetpack Compose
- Room Database
